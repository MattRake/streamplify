Introduction

The Streamplify library offers streams for permutations, combinations and cartesian products.

The source code is available on GitHub.

Getting Started

You need Java 8 or newer in order to use streamplify.

streamplify is available in Maven Central and JCenter.

Maven
<dependency>
    <groupId></groupId>
    <artifactId>streamplify</artifactId>
    <version>1.0.0</version>
</dependency>
Gradle
compile ':streamplify:1.0.0'
